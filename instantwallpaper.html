Help! Sometimes the download button doesn't show (or disappears, or doesn't work)!
This is due to the image hosting site blocking cross origin requests and there is nothing I can do about it.
Right click and select Save Image As... to save the image.
Note: This image saves in the resolution defined above, regardless of how it is displayed below.

Use the button to switch the width and height.

Use query tags img w and h to preset fields.
e.g. standen.link/instantwallpaper.html?img=http://i.imgur.com/JblFDIS.jpg&w=1080&h=1920

The image below is generated by scaling the original image to fill the requested container size.
Use the Grow and Shrink buttons below to modify the image scale within the container.
